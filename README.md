# E-Commerce Backend Platform

Backend e-commerce platform built with ASP.NET Core following Clean Architecture principles.

## Features
- JWT Authentication
- Role-based Authorization
- Product & Category CRUD APIs
- ASP.NET Identity Integration
- Repository & Unit of Work Pattern
- SQL Server + EF Core

## Technologies
- ASP.NET Core
- Entity Framework Core
- SQL Server
- JWT
- Swagger

## Architecture
The project follows Clean Architecture with separation between:
- Domain
- Application
- Infrastructure
- API layers

## Status
Currently implementing payment integration and cloud deployment.

## Running the Project
1. Clone repository
2. Configure database connection
3. Run migrations
4. Start API
